# flutter_book_list

My First Flutter Demo APP

## Getting Started

UI 구현 연습에 중점을 둔, 도서 목록 앱 만들기.

1. 화면

- ListScreen(목록 화면)
- DetailScreen(상세 화면)

2. 기능

- (UI) 도서 목록
- (UI) 도서 상세
- 도서 목록에서 도서를 선택하면 해당 도서 상세 화면으로 이동

3. 모델

- Book
  1. title: 제목
  2. subtitle: 부제
  3. description: 설명
  4. image: 대표 이미지
